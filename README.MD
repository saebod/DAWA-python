# DAWA-python

DAWA-Python bruges til at indsamle oplysninger om boliger i Danmark. Ligenu har jeg primæret forkuseret på **lejligheder** da jeg selv skal bruge det til et projekt.
Scriptet sender en request til [Danmarks Adressers Web API](https://dawa.aws.dk/ "Danmarks Adressers Web API") (DAWA) med adressen hvor der retuneres nedestående oplysninger. 

**Bemærk**
For at få scriptet til at virke skal du bruge **main.py**  filen hvor input skal være en CSV fil der har en kolonne der hedder **adresse**
